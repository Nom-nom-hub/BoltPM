Quickstart

Install

cargo install boltpm

Initialize Project

boltpm init

Add a Package

boltpm install lodash

Usage

boltpm install
boltpm run <script>
boltpm update
boltpm remove <package>
boltpm plugin list

See the full documentation for advanced usage, workspace setup, and plugin development.

Plugins

BoltPM supports both native and WASM plugins. Easily extend your workflow or integrate with other tools.

  • Scaffold: boltpm plugin init myplugin
  • Build: boltpm plugin build
  • Run: boltpm plugin run myplugin

See plugin docs for more.
